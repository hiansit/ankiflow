# AnkiFlow
流れるように学べる、ブラウザベースの汎用暗記カードアプリ。

## デモ
実際にアプリを試すにはこちら:  
[https://hiansit.github.io/ankiflow/](https://hiansit.github.io/ankiflow/)

## 特徴
* **自動読み上げ (TTS)**: 画面を見なくても、音声で学習を進めることができます。
* **マルチデバイス対応**: PC、スマホ、タブレットなど、ブラウザがあればどこでも動作します。
* **カスタマイズ可能**: データを入れ替えることで、様々な科目の学習に使えます。

## 開発について
このプロジェクトは、AIアシスタント（Google Gemini）の支援を受けて開発されました。
